# Product Guide

## Initial Concept
A collaboration hub for project management and workflow automation.

## Vision
The project aims to provide a unified, **workflow-first** collaboration hub for **Feishu-native teams**. It bridges the gap between structured project data (Convex) and everyday communication tools (Feishu), offering a seamless experience for managing complex, multi-step project flows.

## Target Audience
- **Feishu-native teams**: Teams whose primary communication and task management happen within Feishu/Lark.
- Project managers and developers who require more formal **workflow automation** and **approval gates** than basic chat-based tasking provides.

## Goals
- **Workflow Automation**: Transform manual processes into automated, trackable sequences of actions and approvals.
- **UI/UX Excellence**: Provide a top-tier, interactive web interface (Next.js) for high-performance project management.
- **Integrated Ecosystem**: Deeply integrate project data with Feishu's chat, task, and notification systems.

## Key Features
- **Interactive Boards**: Visualizing work items and project stages with high-performance drag-and-drop interfaces.
- **Approval Gates**: Formal sign-offs for critical project milestones, ensuring quality and compliance across the workflow.
- **Feishu Integration**: Deep connection to Feishu for notifications, task syncing, and potentially in-app experiences.
