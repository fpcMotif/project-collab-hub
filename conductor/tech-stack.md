# Technology Stack

## Core Technologies

- **Language**: [TypeScript](https://www.typescriptlang.org/) - For type-safe development across the entire monorepo.
- **Runtime & Package Management**: [Bun](https://bun.sh/) - High-performance JS runtime and package manager.
- **Monorepo Management**: [Turbo](https://turbo.build/repo) - Optimized build system for monorepos.

## Frontend

- **Framework**: [Next.js (App Router)](https://nextjs.org/) - Modern React framework with server-side rendering and efficient routing.
- **UI Library**: [React 19](https://react.dev/) - The latest stable version of React.
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) - Utility-first CSS framework for rapid and modern UI development.

## Backend & Data

- **Data Platform**: [Convex](https://www.convex.dev/) - Real-time database and backend logic platform for seamless data syncing.
- **Functional Programming**: [Effect](https://effect.website/) - Library for building robust and scalable applications with functional programming patterns.

## Integrations

- **Communication Hub**: [Feishu/Lark Integration](https://www.feishu.cn/) - Deep integration with Feishu for notifications, tasks, and workflow automation.

## Quality & Tooling

- **Linting & Formatting**: [oxlint](https://oxc.rs/docs/guide/usage/linter.html), [oxfmt](https://oxc.rs/docs/guide/usage/formatter.html), [Ultracite](https://github.com/steipete/ultracite) - High-performance linting and formatting tools.
- **Testing**: [Vitest](https://vitest.dev/) (Unit/Integration) and [Playwright](https://playwright.dev/) (E2E).
- **Git Hooks**: [Lefthook](https://github.com/evilmartians/lefthook) - Fast and powerful Git hooks manager.
